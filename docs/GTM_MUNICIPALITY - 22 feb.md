## Säljprocess B2G

### Typisk säljcykel: 6-18 månader

```
Vecka 1-4:    Identifiera kontaktperson (arbetsmarknadschef/nämnd)
Vecka 4-8:    Första möte (demo, problemvalidering)
Vecka 8-16:   Pilot-förslag (gratis eller rabatterad 3-mån pilot)
Vecka 16-24:  Pilot genomförs, feedback samlas
Vecka 24-36:  Utvärdering, budgetprocess, beslut
Vecka 36-52:  Upphandling/avtal, onboarding
```

### Beslutsfattare

| Roll | Funktion | Vad de bryr sig om |
|------|----------|-------------------|
| Arbetsmarknadschef | Operativ ledning av arbetsmarknadsinsatser | Resultat, effektivitet |
| Kommunstyrelse | Strategiska beslut, budget | Kostnadsbesparingar, politiskt kapital |
| IT-chef | Teknisk utvärdering, integration | Säkerhet, GDPR, driftskostnad |
| Kommunchef | Övergripande prioriteringar | Kommunens attraktivitet, tillväxt |

### Ingångsstrategi: Arbetsmarknadschefen

De flesta kommuner har en arbetsmarknadschef eller -nämnd som ansvarar för lokala insatser. Det är den mest naturliga första kontakten.

## Piloterbjudande

### Gratis 3-månaders pilot

| Leverans | Beskrivning |
|----------|-------------|
| CityIQ-rapport | Talangbristanalys för kommunen (vilka yrken saknas?) |
| ROI-kalkyl | Vad kostar det att inte agera? Potentiell skatteintäkt vid minskad arbetslöshet |
| TalentFlow-åtkomst | Gratis för 100 invånare under pilotperioden |
| Utvärderingsrapport | Sammanfattning efter 3 månader |

### Konvertering pilot → betalande

**Mål**: 60-70% konverteringsgrad

Strategi:
1. Under piloten, visa konkret värde med siffror
2. "Om 50 invånare byter till bristyrken = X kr mer i kommunalskatt"
3. Jämför med kostnad för alternativ (konsulter: 500k-2M per projekt)

## Upphandling

### Direktupphandling (<700 000 SEK)

Kommuner får göra direktupphandling under tröskelvärdet (ca 700k SEK). Vår prissättning (75-250k/år) är **medvetet under detta tak** — det innebär:
- Inget formellt upphandlingsförfarande behövs
- Arbetsmarknadschefen kan ofta besluta själv eller via nämnd
- Snabbare process (veckor istället för månader)

### Formell upphandling

Om kommunen kräver upphandling:
- Vi behöver vara registrerade som leverantör
- Följa LOU (Lagen om offentlig upphandling)
- Skriva anbudsunderlag
- **Rådgivare med upphandlingserfarenhet är kritisk**

### SKR / Adda

**Adda** (fd SKL Kommentus) är kommunernas gemensamma inköpscentral. Om vi blir Adda-godkänd leverantör:
- Alla 290 kommuner kan köpa utan egen upphandling
- Massivt förenklad säljprocess
- **Mål för År 2**: Bli Adda-leverantör

## Marknadsföring mot kommuner

### Kanaler

| Kanal | Kostnad | Effektivitet |
|-------|---------|-------------|
| Direktkontakt (LinkedIn/mejl/telefon) | Tid | Hög |
| Science Park-nätverk | Gratis | Medium-Hög |
| SKR-konferenser (Kommek etc) | 5-20k SEK | Hög |
| Kommun-specifika case studies | Tid | Mycket hög |
| Vinnova-finansiering som kvalitetsstämpel | 0 (om beviljad) | Hög |

### Nyckelbudskap

1. "Ni spenderar X kr/år på arbetsmarknadsåtgärder — vi visar vad som faktiskt fungerar"
2. "Era invånare får ett gratis karriärverktyg som visar dolda jobbmöjligheter"
3. "Bättre matchning = kortare arbetslöshetsperioder = mer skatt till kommunen"
4. "75k SEK/år istället för en konsultrapport på 1M SEK"

## Risker

| Risk | Hantering |
|------|-----------|
| Lång säljcykel (>12 mån) | Gratis piloter för att accelerera, Vinnova som brygga |
| Kommuner har inte budget | Positionera som kostnadsbesparing, jämför med RoM-utgifter |
| Personalbyte i kommunen | Bygga relationer med flera kontakter per kommun |
| Upphandlingshinder | Prissättning under direktupphandlingsgräns, Adda-avtal |
| Politiska val (2026) | Partipolitiskt neutral positionering |
